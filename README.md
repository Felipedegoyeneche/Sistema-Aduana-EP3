# Sistema Aduana EP3 — RQY1102 Ingeniería de Software

## Descripción
Prototipo funcional de un sistema de gestión aduanera para el paso fronterizo terrestre Cardenal Samoré (Chile-Argentina), desarrollado como parte de la Evaluación Parcial N°3 de la asignatura Ingeniería de Software (RQY1102) — DuocUC 2024.

## Problema abordado
Los pasos fronterizos terrestres chilenos presentan esperas de 8 a 20 horas por falta de digitalización. Este sistema automatiza los procesos de documentación de menores, trámites de vehículos, revisiones SAG y PDI, e integración entre aduanas limítrofes.
